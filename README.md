抽象的、渋谷スクランブル PWA


ネオンがきらめく夜の渋谷交差点を再現したインタラクティブなウェブアート。  
満月の夜空、信号に従って動く車と歩行者、そしてビルに灯るネオンサインが、都市のリズムを描き出します。

---

特徴

- フルスクリーン対応のPWA（Progressive Web App）
- 信号制御に合わせて車と人が動くリアルな交差点シミュレーション
- ランダム生成されるネオンビルと星空
- レスポンシブデザインでスマホ・タブレット・PCに最適化

---
 インストール方法

1. GitHub Pagesで公開されたURLにアクセス  
2. スマートフォンのブラウザで「ホーム画面に追加」  
3. アプリのように起動して楽しめます！

---
 開発環境

- HTML / CSS / JavaScript
- PWA構成：`manifest.json` + `sw.js`（Service Worker）
- GitHub Pagesでホスト
